# 뉴스레터 키워드 검색 기준

---

## 01. 검색 쿼리 규칙

- 쿼리에 반드시 `2026` 또는 `최근` 포함
- 키워드 검색과 편의점 신상 검색을 동시에 수행

### 쿼리 템플릿

```
{키워드} 편의점 2026
{키워드} CU 신상 2026
{키워드} GS25 신상 2026
{키워드} 세븐일레븐 신상 2026
{키워드} 이마트24 신상 2026
{키워드} CU
{키워드} GS25
{키워드} 세븐일레븐
{키워드} 이마트24
```

> 신상 쿼리는 새로 출시된 제품 탐색용.
> 편의점명만 단독으로 검색하는 쿼리는 기존 입점 제품이 뜨는 경우를 위한 것.

## 02. 유관 키워드 함께 검색

키워드 하나만 검색하지 말고 연관 키워드를 함께 검색.

| 키워드 예시 | 유관 키워드 |
|------------|-----------|
| 두쫀쿠 | 피스타치오, 카다이프 |
| 황쫀쿠 | 황치즈, 카다이프 |
| 콘피쵸 | 콘쵸, 피스타치오, 크라운 |
| 흑당 | 버블티, 타피오카 |
| 말차 | 녹차, 그린티 |
| 망곰이 | 망그러진 곰, 콜라보, 띠부씰 |

## 03. 데이터랩 사용 주의사항

배지 판정용 트렌드 점수는 `fetch_naver_trend.py`로 떡볶이 기준선 비교 (이전 v6 가이드 그대로 유지: HOT 30%+, 주목 10~29%, 초기 3~9%).

**반드시 지킬 것:**
1. **단독 키워드 일별(date) 조회**만 사용 — 주간(week) 금지
   - 주간 조회는 일별 피크가 뭉개져서 신호가 약해 보임
2. **5개 이상 묶음 비교 금지** — 작은 키워드가 큰 키워드에 의해 0으로 찌부러짐
3. **떡볶이를 모든 배치에 앵커로** 포함 (5개씩 묶을 때 첫 자리)
4. **떡볶이 비교는 점주에게 노출 금지** — 내부 판단 도구

## 04. SNS 애칭 → 정식 명칭 검증 (★ 핵심)

SNS에서 본 키워드가 정식 상품명이 아닐 수 있음. 반드시 검증.

### 검증 채널
| 채널 | 용도 |
|------|------|
| 일반 웹 검색 | 기본 |
| 펴늬 (pyony.com) | 편의점 행사상품 DB |
| 네이버 블로그 | 후기·내돈내산 |
| 인스타그램 | 신상 인증 |
| 더쿠 / 인스티즈 | 커뮤니티 후기 |
| 제조사 공식 사이트 | 정식 명칭 |
| 공식 보도자료 | 출시일·정확 정보 |

### SNS 줄임말 패턴 (검증 사례)
| SNS 애칭 | 정식 명칭 | 패턴 |
|---------|----------|------|
| 두쫀쿠 | 두바이 쫀득 쿠키 (편의점별 라인업) | 단어 첫글자 |
| 황쫀쿠 | CU 단짠 황치즈 쫀득쿠키 | 황치즈+쫀득쿠키 줄임말 |
| 콘피쵸 | 크라운 콘쵸 + 피스타치오 한정판 | 콘쵸+피스타치오 줄임말 |
| 니모 아이스크림 | GS25 프로즌 피쉬 소르베 | 비주얼 별칭 |

### 본문 표기
- "(SNS 애칭 'XX')"로 명시
- 검증 실패 시: 임의 매핑 금지, 사용자에게 정식명 확인 요청
