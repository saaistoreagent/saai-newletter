# SAAI 편의점 뉴스레터 — Voice & Tone 가이드

> 편의점 점주가 읽고 1분 안에 행동하게 만드는 글쓰기 원칙

---

## 01. 타겟 독자 퍼소나

> 네이버 카페 편의점 점주 커뮤니티 투표 결과(n=149) 기반.

| 연령대 | 득표 | 비율 |
|--------|------|------|
| 20대 | 12표 | 8.1% |
| **30대** | **38표** | **25.5%** |
| **40대** | **56표** | **37.6%** |
| 50대 | 26표 | 17.4% |
| 60대 이상 | 17표 | 11.4% |

→ **30~40대가 전체의 63.1%** 로 핵심 독자층

### 메인 퍼소나 — "바쁜 40대 점주 김사장"
- 오전 발주 → 매장 관리 → 저녁 재고 정리. 스마트폰으로 빠르게 정보 소비
- 핵심 니즈: "트렌드 상품 놓치지 않고 먼저 발주하고 싶다"
- 핵심 불만: "읽어도 뭘 해야 할지 모르겠는 글은 시간 낭비"

### 서브 퍼소나 — "트렌드 민감한 30대 점주 이사장"
- 인스타·틱톡으로 트렌드 파악, SNS 반응 보고 발주 결정
- 핵심 니즈: "SNS에서 뜨기 전에 먼저 발주해서 초도 물량 잡고 싶다"

### 독자의 핵심 질문
> **"그래서 당장 뭘 발주하고, 뭘 빼야 하는 거야?"**

## 02. 보이스 톤

**정중함 × 단호함 × 실용성**

- 독자를 존중하되 불필요한 감성 표현 배제
- 추측이 아닌 검색된 사실만 사용
- 모든 문장의 끝은 반드시 '지금 당장 할 수 있는 행동'으로 마무리

---

## 03. 어미 규칙

✅ 허용: `~해요` / `~하세요` / `~예요` / `~있어요`

❌ 금지: `~합니다` / `~임` / `~해` / `~야` / `~잖아` / `~할 것 같아요`

## 04. 금지 표현

| 구분 | ❌ 금지 | ✅ 대체 |
|------|--------|--------|
| 인사말 | 안녕하세요! 이번 주도 파이팅입니다. | (인사말 없이 바로 본론으로) |
| 감성 표현 | 점주님들의 노고에 감사드려요. | (생략) |
| 모호한 추측 | ~해보는 건 어떨까요? | 발주 리스트에 추가하세요. |
| 추측성 워딩 | 잘 팔릴 것 같아요. | 검색량이 상승 중이에요. |
| 수치 지어내기 | 매출이 23% 상승했어요. (근거 없음) | (수치 없이 사실만) |
| 업계 용어 | HMR | 즉석식품 |
| 업계 용어 | 객단가 | 1인당 구매 금액 |
| 업계 용어 | RTD커피 | 캔커피 |
| 업계 용어 | 온장 음료 | 따뜻한 음료 |
| 업계 용어 | 냉장 진열 | 차가운 음료를 앞쪽에 |
| 업계 용어 | 크로스셀링, 업셀링, 투 트랙, 시너지 | 쉬운 말로 풀어쓸 것 |
| 어색한 표현 | 수요가 붙는 흐름이에요. | 찾는 손님이 늘고 있어요. |
| 어색한 표현 | 반응이 붙을 수 있어요. | 여러 개 사는 손님이 많아요. |
| 어색한 표현 | 낮 유동이 길어져요. | 낮에 손님이 더 많아져요. |
| 어색한 표현 | 나들이형 수요 | 나들이 손님 |
| 어색한 표현 | 도심형 상권은 체류 수요가 몰려요. | 광화문 인근 매장은 손님이 한꺼번에 몰릴 수 있어요. |
| 어색한 표현 | 미입점 품목은 보수적으로 보세요. | 아직 편의점에 없으니 입점 소식 주시하세요. |
| "이번 주" 표현 | 이번 주 트렌드 (본문 중) | 4월 1주차 트렌드 (단, 섹션 제목 "이번주 편의점 트렌드"는 허용) |

## 05. CTA 규칙

CTA는 아래 방향을 참고하되 문맥에 맞게 자연스럽게 작성. 기계적으로 붙여넣기 금지.

| 상태 | CTA 방향 |
|------|---------|
| 상승 | 발주 확대 / 진열 강화 / 행사 확인 등 구체적 행동 |
| 유지 | 발주 유지 / 재고 점검 |
| 하락 | 추가 발주 주의 / 재고 소진 집중 (신상 정보 추가 금지) |
| 메가트렌드 | "입점 소식을 주시하세요" 패턴 |
| 날씨 | 구체적인 진열/발주 행동 (KBO 경기장, 축제 등 구체적 장소 언급) |
| 이벤트 | 발주량 점검 / 확대·축소 / 상권별 대비 |

## 06. 편의점 신상 언급 규칙

- CU · GS25 · 세븐일레븐 · 이마트24 신상/입점 여부 반드시 검색
- 확인된 경우만 기재. 추측으로 제품명 생성 절대 금지.
- 하락 키워드에는 신상 정보 미기재
- **가격 일관성**: 가격을 넣으려면 해당 호의 모든 상품에 넣을 것. 일부만 넣고 일부는 빼지 않기. 확인 안 된 가격은 "약"을 붙이지 말고 아예 빼기.
- **행사 정보**: 1+1, 2+1 등 행사가 확인되면 반드시 기재. 점주에게 가장 실용적인 정보.

## 07. 데이터 표현 규칙

`가설검증_통합결과.xlsx` (오피스 상권 매출 데이터)를 날씨·이벤트 TIP에 활용할 때:

| ❌ 금지 | ✅ 대체 |
|--------|--------|
| 냉음료 매출이 24.1% 증가해요 | 냉음료가 눈에 띄게 늘어요 |
| 공휴일 영수증수가 57% 줄어요 | 공휴일에 방문 고객이 크게 줄지만 |
| 객단가가 19% 올라요 | 1인당 구매 금액은 올라가는 경향이 있어요 |
| 맑은 날 매출 +7.2% | 맑은 날은 매출이 더 높은 경향이 있어요 |

데이터는 근거로 활용하되, 숫자를 직접 노출하지 않고 자연어로 표현.
